## Controls

| **Key**             | **Action**             |
| ------------------- | ---------------------- |
| `W`                 | Move Forward           |
| `S`                 | Move Backward          |
| `A`                 | Turn Left              |
| `D`                 | Turn Right             |
| `P`                 | Pause/Unpause the game |
| `Spacebar`          | Drop a depth charge    |
| `Left Mouse Button` | Fire a bullet          |
| `Mouse Movement`    | Aim the turret         |
| `Esc`               | Exit pointer lock      |
